## Usage Guidelines

### Size Recommendations

Material Icons uses size classes like `md-18`. For emoji replacements:

```css
.emoji-icon {
  font-size: inherit; /* Inherit from parent */
  vertical-align: middle;
  display: inline-block;
}

.emoji-icon.size-14 { font-size: 14px; }
.emoji-icon.size-16 { font-size: 16px; }
.emoji-icon.size-18 { font-size: 18px; }
.emoji-icon.size-20 { font-size: 20px; }
.emoji-icon.size-24 { font-size: 24px; }
.emoji-icon.size-36 { font-size: 36px; }
```

### Color Considerations

- **Emoji**: Have fixed colors (cannot be styled with CSS color)
- **Unicode symbols**: Inherit text color (can be styled)
- **Recommendation**: Use Unicode symbols for status indicators that need color coding

### Accessibility

Always provide accessible labels:

```html
<!-- Good -->
<span aria-label="Warning" role="img">âš ï¸</span>

<!-- Better -->
<span class="emoji-icon" aria-label="Warning" role="img" title="Warning">âš ï¸</span>

<!-- With fallback -->
<span class="emoji-icon" role="img" aria-label="Warning">
  <span aria-hidden="true">âš ï¸</span>
  <span class="sr-only">Warning</span>
</span>
```
